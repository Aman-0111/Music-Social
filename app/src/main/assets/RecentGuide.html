Recent Songs Page Information


The recent songs page contains a display button that appears after the users seven most recent songs played are obtained. The button displays these songs as buttons for the user to tap.

Before Button Press

After Button Press



Pressing these buttons will have different results based on whether Spotify is installed or not. If Spotify is not installed than the song will be opened in the browser. On the other hand, if Spotify is installed than a song player will start the song. In both cases a notification is sent.

Without Spotify



With Spotify



Please note that Spotify only allows users with premium memberships to play specific songs. Therefore, if a user without premium presses a previous track instead of playing the specific track Spotify will play recommended tracks. This is something that Spotify have implemented and cannot be changed.
