Home Page Information


The Home page within the application will always look the same whether Spotify is installed or not. However, the result of tapping on an artists profile will change.


Before Button Press

After Button Press



The display button will show after the app loads some random Artists from Spotify. Then pressing the button will display the image for these Artists as recommendations.


With Spotify

Without Spotify



If the user has Spotify the artists image will redirect the user to the artists page on Spotify. Without Spotify the users browser will open and the web version of Spotify will open on the artists page.
