Login Information


When you open the Music Social you will be greeted by this page below.




To sign in you must use an account that was registered with the app. Register an account the register text can be tapped to open the register page. The register page can be seen below.




To register an account, add a username and a password of minimum length 6. The default eee account is an exception to this rule. When logging in with the default account eee can be used as the username and password without consequences. After logging in the user will be prompted to Login into Spotify. There are differences based on whether the Spotify app is installed or not.


With Spotify Installed

Without Spotify



If Spotify is installed the app will use the Spotify account connected to the Spotify app and log in. If not logged into the app the user will be redirected to the Spotify app to login (image could not be obtained due to privacy policies). When Spotify is not installed the user will be prompted to login in manually within the app.
